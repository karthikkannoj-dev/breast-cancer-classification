# Breast Cancer Classification using SVM

## Objective
To classify tumors as Malignant or Benign using Support Vector Machine.

## Dataset
Breast Cancer dataset from sklearn.

## Steps
- Data loading
- Feature scaling
- SVM with Linear & RBF kernel
- Model evaluation
- ROC-AUC analysis

## Results
- Accuracy: ~98%
- Best Kernel: RBF
- AUC: ~0.99

## Tools Used
Python, Scikit-learn, Matplotlib, Seaborn
